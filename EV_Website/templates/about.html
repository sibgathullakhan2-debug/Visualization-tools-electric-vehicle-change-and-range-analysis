About This Project

This project was developed as part of the Skill Wallet – Data Analytics with Tableau program. It focuses on analyzing electric vehicle data and presenting meaningful insights using interactive dashboards, visual storytelling, and a Flask-based web interface.

🎯 Objective

Analyze EV market trends including vehicle range, efficiency, charging speed, body styles, and pricing.
