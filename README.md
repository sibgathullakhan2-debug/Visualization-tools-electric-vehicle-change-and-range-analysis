# ⚡ Electric Vehicle Market Analysis Dashboard

A web-based data visualization project developed using **Tableau**, **MySQL**, **Python**, and **Flask** as part of the **Skill Wallet - Data Analytics with Tableau** program.

---

## 📌 Project Overview

This project analyzes electric vehicle market trends and presents interactive insights through Tableau dashboards integrated into a Flask web application.

The project helps users understand:

- EV brand performance
- Vehicle efficiency
- Driving range
- Fast charging capability
- Price distribution
- Body style distribution

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone <your-github-repository-link>
```

### 2. Open the project

```bash
cd EV_Website
```

### 3. Install dependencies

```bash
py -m pip install -r requirements.txt
```

### 4. Run the application

```bash
py app.py
```

### 5. Open in browser

```
http://127.0.0.1:5000
```
